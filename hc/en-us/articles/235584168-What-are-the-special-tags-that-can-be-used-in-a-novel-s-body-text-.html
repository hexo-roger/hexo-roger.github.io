What are the special tags that can be used in a novel's body text?

▶On mobile

▶On desktop

 

On mobile

You can add special tags on the mobile version by going to the Novel posting page, unfurling Tools above the body text entry field, and then selecting the tag you'd like. (This is applicable to Android users and users of iOS13 and above.)

 

■ Novel layout


When tapped, you'll be able to select "Add chapter title" and "Add a page".

  • Add chapter title
    [chapter:Chapter Title]
    This tag is used to create chapter titles, which have larger and bolder lettering. By adding chapter titles to your work, they'll also be displayed in the table of contents for the work which allows easy transitioning between sections.
  •  Add a page
    [newpage]
    This tag will make the following text show up on another page.
    Click here for more details on [newpage]

■ Add an image


When tapped, you'll be able to select "Upload images" and "Add images via work ID".
"Upload images" lets you directly upload one or more images to add to a Novel's text. For more details, please refer to the "What is the Upload Illustrations feature?" article.
"Add images via work ID" lets you add illustrations and manga works to your novel by specifying their ID on pixiv.

  • Upload images
    [uploadedimage:UploadedIllustrationID]
    You can upload illustration images directly using the Tool menu. One novel work can upload a maximum of 30 images.
    For more information, please refer to the "What is the Upload Illustrations feature?" article.

  • Add images via work ID
    [pixivimage:Work ID]
    You can add illustration and manga work images already on pixiv to the body of your novel by using this tag in your work. If you set a work that was uploaded by another creator, you will need to wait until they grant permission for their work to be used in yours. You can use up to 300 illustrations in one novel work.

    If you want to use an illustration or manga work with multiple images in it, you'll need:
    [pixivimage:Work ID-▲]
    The ▲ portion of this tag represents the page number of the illustration or manga work with multiple images.

■ Add URL

When tapped, you'll be able to select "Link a page within the novel" and "Add a URL link".
You can use "Link a page within the novel" to put links to other pages in your novel. "Add a URL link" is used to link to other works or external sites.

  • Link a page within the novel
    [jump: Link destination page number]
    You can specify within the text to move to another page within the novel if it has more than one page. Please feel free to use it to create a Table of Contents or even to create interactive, choice-based stories.
  • Add a URL link
    [[jumpuri: Title > Link destination URL]]
    You can link a novel to another novel by specifying an optional work URL. You can also link to websites that aren't pixiv.

■ Add ruby characters

[[rb:pixiv > ピクシブ]]
You can add pronunciation notations (called furigana in Japanese, called ruby characters in typeset terminology) in hiragana, katakana, kanji, and alphabet characters. Example: you can have ピクシブ as notations over "pixiv".
