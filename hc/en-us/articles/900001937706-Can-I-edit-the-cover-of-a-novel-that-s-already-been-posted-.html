Can I edit the cover of a novel that's already been posted?

On desktop

If you uploaded your own image

Please upload the image you want to reuse from the upload cover section of the novel posting page.

 

If you used a cover template

Please edit and make adjustments to the template, as well as choose a new cover from among the existing templates.
* Even if you'd like to use the same template, you will have to select it again.

 

On mobile

If you uploaded your own image

Please upload the image you want to reuse from the upload cover section of the novel posting page.

 

If you used a cover template

Please edit and make adjustments to the template, as well as choose a new cover from among the existing templates.
* Even if you'd like to use the same template, you will have to select it again.
