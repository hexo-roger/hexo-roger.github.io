pixiv프리미엄에서 이용하는 신용카드를 삭제・변경하고 싶어요

  • pixiv프리미엄의 결제방법으로 등록한 신용카드를 삭제하려는 경우
    이쪽에서 등록된 신용카드의 확인과 삭제가 가능합니다.
    ※pixiv프리미엄의 유효기한이 다 되어 차회 갱신 처리가 진행중이거나, (청구 이전의)지불되지 않은 결제가 있는 경우 신용카드를 삭제할 수 없습니다.

 

  • pixiv프리미엄의 결제방법으로 등록한 신용카드를 변경하려는 경우
    등록된 플랜에 따라 아래의 순서대로 신용카드를 변경해 주시기 바랍니다.

    1개월 플랜으로 등록한 경우 :
    〈프리미엄 등록 정보 확인 페이지〉 → 결제방법 항목에 표시된 〈변경〉 버튼 → 새로 이용하려는 카드 정보 입력 → 〈변경〉버튼 → 변경 완료

    3·6·12개월 플랜을 이용하는 경우 :
    〈프리미엄 등록 정보 확인 페이지〉 → 결제방법 항목에 표시된 〈변경〉 버튼 → 새로 이용하려는 카드를 추가 혹은 선택 → 〈선택한 카드 사용하기〉버튼 → 변경 완료

    ※pixiv 프리미엄으로 등록된 후 결제방법으로 사용하는 신용카드를 변경한 경우, 다음 pixiv프리미엄 갱신일부터 청구 대상이 변경됩니다.
