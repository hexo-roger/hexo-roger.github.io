pixiv에 일러스트를 투고하는 방법을 알고 싶어요

pixiv 계정으로 로그인한 후 ‘작품 투고’ 메뉴에서 투고할 수 있습니다.
※투고하는 작품은 자신이 제작하고 저작권을 가진 것에 한정합니다. 타인의 작품을 무단으로 투고하는 등의 행위는 이용 규약에 위배되니 삼가해 주시기 바랍니다.


▶PC 버전
▶모바일 버전
▶앱 버전


PC 버전

  1. 화면 상단의 ‘작품 투고’ 에서 ‘일러스트’를 클릭합니다.


  2. ‘파일 선택’을 클릭합니다.

  3. 투고하고 싶은 파일을 선택합니다. 키보드의 ‘SHIFT’ 버튼을 누르면서 파일을 선택하거나, 마우스를 드래그하여 여러 장의 파일을 선택할 수 있습니다. 선택 후 ‘열기’를 클릭합니다.
    ※업로드할 파일을 여러 개 선택할 수 있습니다.
    ※파일 크기는 1장당 32MB 이하로 제한되며, 파일 형식은 JPG, GIF, PNG를 지원합니다. 애니메이션 GIF는 지원하지 않습니다.

    여러 장을 투고하는 경우, 프리뷰에서 섬네일 이미지를 드래그하여 순서를 바꿀 수 있습니다.
    그리고 프리뷰의 ‘섬네일 조정하기’에서 섬네일의 표시 위치를 조정할 수 있습니다.

  4. 아래의 항목은 입력·선택 모두 필수입니다.
    - 태그
    - 연령제한
    기타 항목들도 필요에 따라 입력·선택해 주세요.

  5. 화면 하단의 ‘투고하기’를 클릭하면 투고가 완료됩니다.
    확인 화면에서 ‘OK’를 클릭하면 작품이 투고됩니다.


  6. 투고가 올바르게 완료되면 ‘작품이 투고됐습니다’라고 표시됩니다.


모바일 버전

  1. [ 三 ] 메뉴 버튼을 탭 합니다.


  2. 메뉴 안에서 ‘투고하기’를 탭 한 다음 ‘일러스트’나 ‘만화’를 선택합니다.

  3. ‘이미지 추가’에서 투고하고 싶은 파일을 선택합니다. 프리뷰의 ‘섬네일 조정하기’에서 섬네일의 표시 위치를 조정할 수 있습니다.
    ※업로드할 파일을 여러 개 선택할 수 있습니다.
    ※파일 크기는 1장당 32MB 이하로 제한되며, 파일 형식은 JPG, GIF, PNG를 지원합니다. 애니메이션 GIF는 지원하지 않습니다.

  4. 아래의 항목은 입력·선택 모두 필수입니다.
    - 태그
    - 연령제한
    기타 항목들도 필요에 따라 입력·선택해 주세요.

  5. 화면 하단의 ‘투고하기’를 탭 하면 투고가 완료됩니다.
    투고가 올바르게 완료되면 ‘작품이 투고됐습니다’라고 표시됩니다.

앱 버전

  1. ‘마이 페이지’에서 ‘작품 투고’→‘일러스트’를 선택합니다.

  2. 이미지 폴더가 열리면 투고하고 싶은 파일을 순서대로 선택합니다(선택한 이미지는 화면 상단에 순번대로 표시됩니다).

  3. 태그, 연령제한 설정, 성적 묘사 등 필수 항목을 입력하고 화면 하단의 ‘투고하기‘를 선택하면 투고가 완료됩니다.
    ※업로드할 파일을 여러 개 선택할 수 있습니다.
    ※파일 크기는 1장 당 32MB 이하로 제한되며, 파일 형식은 JPG, GIF, PNG를 지원합니다. 애니메이션 GIF는 지원하지 않습니다.

투고가 올바르게 완료되면 투고한 작품의 열람화면이 표시됩니다.

그리고 재투고의 경우, pixiv프리미엄 유저에 한해 PC버전이나 모바일 버전을 통해 진행할 수 있습니다.
