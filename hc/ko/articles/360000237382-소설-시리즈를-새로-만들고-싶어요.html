소설 시리즈를 새로 만들고 싶어요

▶PC 버전
▶모바일 버전
▶앱 버전

PC 버전

■소설 투고 페이지에서 작성하기

  1. ‘작품 투고’ -> ‘소설’ -> ‘시리즈 작품’을 클릭합니다.
  2. ‘시리즈’ 항목이 나오면 클릭하여 시리즈 목록을 표시합니다.
  3. 시리즈 목록 맨 아래에서 ‘시리즈 작성’을 클릭하면 팝업창이 나옵니다.
  4. 시리즈 타이틀 등을 입력·선택하고 ‘저장하기’를 클릭하면 새 시리즈가 추가되어 작품에 적용됩니다.

■대쉬보드에서 작성하기

  1. 아이콘 메뉴에서 ‘대쉬보드’를 클릭합니다.
  2. ‘작품’ → ‘소설’ → ‘시리즈’로 이동합니다.
  3. 화면의 ‘시리즈 작성’을 클릭하면 팝업창이 나타납니다.
  4. 시리즈 타이틀 등을 입력・선택하고 ‘저장하기’를 클릭하면 새 시리즈가 추가되어 작품에 적용됩니다.

 

모바일 버전

■소설 투고 화면에서 작성하기

  1. [ 三 ] 메뉴의 ‘투고하기’ -> ‘소설’ -> 투고 형식의 ‘시리즈 작품’을 탭 합니다.
  2. ‘시리즈’ 항목이 나오면 풀다운 메뉴를 탭 하여 시리즈 목록을 표시합니다.
  3. 시리즈 목록 맨 아래에서 ‘시리즈 작성’을 선택하면 팝업창이 나옵니다.
  4. 시리즈 타이틀 등을 입력·선택하고 ‘변경 내용을 저장’을 탭 하면 새 시리즈가 추가되어 작품에 적용됩니다.

■대쉬보드에서 작성하기

  1. [ 三 ] 메뉴에서 ‘대쉬보드’를 탭 합니다.
  2. ‘작품’ → ‘소설’ → ‘시리즈’로 이동합니다.
  3. 화면 오른쪽에 있는 [ + ]버튼을 탭 하면 팝업창이 나타납니다.
  4. 시리즈 타이틀 등을 입력·선택하고 ‘저장하기’를 클릭하면 새 시리즈가 추가되어 작품에 적용됩니다.
